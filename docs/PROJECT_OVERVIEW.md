# Expense Tracker Application

## Overview
A comprehensive expense tracking system with authentication, detailed transaction management, and customizable reporting.

## Tech Stack
- **Backend**: Java 17 + Spring Boot 3.2
- **Frontend**: React 18 + TypeScript + Vite
- **Database**: PostgreSQL 15
- **Authentication**: Spring Security + JWT
- **UI**: Tailwind CSS + shadcn/ui components

## Features
1. **Authentication**: Username/password login with JWT tokens
2. **Transaction Management**: Track income/expenses with full details
3. **Account Management**: Multiple accounts with custom names
4. **Payment Methods**: UPI, Card, Cash, Bank Transfer tracking
5. **Reporting**: Monthly/daily reports with download capability
6. **Dashboard**: Visual overview of financial status

## Database Schema
- **users**: Authentication and user profile
- **accounts**: Bank accounts with opening balance
- **transactions**: All financial transactions
- **payment_methods**: UPI IDs, card names, etc.
